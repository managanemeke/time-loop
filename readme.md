# time-loop-shooting

"Петля времени" - технический триллер о разработчиках,
случайно создавших реальную временную аномалию.

Команда разработчиков, заваленная дедлайнами,
в шутку придумывает концепцию "петли времени" -
бесконечного повторения одного дня для решения всех задач.

Но шутка становится реальностью - идея обретает
законченную форму, и команда решает открыть петлю...

Оператор, пришедший снимать документалку о работе IT-команды,
становится случайным свидетелем прорыва в четвертое измерение
и единственным помощником в ситуации, где реальность стала
мифом.
